== Description ==

This plugin pings blogolb.net servers everytime you makes a new Blog Post so that blogolb.net can index your content easily and quickly.

<a href=\"http://www.blogolb.net\" target=\"_self\">BloggolB</a> is an attempt to develop a searchable and categorized directory of the thousands of Wordpress Blogs out there.

The number of blogs launching each day is so vast that most of them dont even get a chance to connect to the target audience. What Blogolb intends to do is to provide a realtime and searchable window to all these blogs.

Any wordpress blog can install the BlogolB plugin which will then ping BlogolB each time you make a blog post. Our realtime search and indexing system then gathers and categorizes your blog content and provides visitors with a single window interface to all the content.

Visitors can browse through the whole archive and quickly reach the original website.

**How it helps users**

*	Find blogs which are discussing their area of interest instantly.
*	Quickly search across blogs.
*	Easily follow blogs in their area of interest

**How it helps bloggers**

*	Reach the target audience more effectively.
*	Keep track of similar blogs.
*	Get more targeted traffic.

**How much does it cost**

*	BlogolB is a 100% free service.


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
